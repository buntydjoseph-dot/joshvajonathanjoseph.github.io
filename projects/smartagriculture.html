LoRa + MQTT Smart Agriculture Monitoring & Irrigation System

A distributed IoT-based agricultural automation system designed to overcome real-world field constraints such as long-distance sensor deployment, network limitations, and reliable actuator control.

The system uses a central concentrator (gateway) that bridges wired/local weather sensing and long-range LoRa soil nodes into an MQTT-based communication architecture. The main controller executes irrigation control logic using relays driving solenoid valves, while telemetry and remote commands are synchronised via MQTT.

System Architecture

The concentrator acts as the communication backbone of the farm network:
  • Weather Sensors → Wired/Local → Concentrator
  • Soil Nodes (MCU + LoRa) → LoRa RF → Concentrator
  • Concentrator → MQTT → Main Controller / Cloud
  • Main Controller → Relays → Solenoid Valves
  • Local UI → Direct connection to Main Controller
