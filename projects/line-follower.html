Mini Line Follower Robot with RFID Detection

A compact autonomous robotic system designed to perform accurate line tracking while detecting RFID tags placed along the path.
Project Overview
This project focuses on developing an embedded robotic platform capable of autonomous navigation using optical sensing combined with RFID-based event detection. The system was designed to demonstrate reliable sensor integration, motor control, and real-time decision-making.
Key Components
  • ESP32 Microcontroller (Main Control Unit)
  • LSAA08 Line Following Sensor
  • PN5180 RFID / NFC Reader Module
  • DC Motor Driver
  • Buck Converter for Power Regulation
Technical Highlights
  • Embedded firmware development for multi-sensor interfacing
  • Closed-loop navigation logic for precise line tracking
  • RFID-based conditional behaviour implementation
  • Motor control and system optimisation
  • Hardware integration and debugging
