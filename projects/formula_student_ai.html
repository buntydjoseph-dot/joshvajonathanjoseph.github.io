Demo 2 — YOLO Cone Detection + Client–Server Control

Developed a deep learning model using YOLO, trained on a cone dataset for detection. The model runs on a PC and uses a client–server architecture to send control commands from the PC to the vehicle system.
  • Deep learning model: YOLO (cone detection)
  • Training: cone dataset
  • Runtime: model inference executed on PC
  • Control: commands sent via client–server architecture

Key Contributions

  • Built an autonomous navigation workflow using QLabs simulation and physical QCar testing.
  • Implemented computer vision for object identification and validated behaviour on the physical platform.
  • Developed a YOLO-based cone detection model and integrated it with a PC-based inference + client–server control pipeline.
