Simulation-Based Autonomous Navigation (QLabs)

Developed and validated autonomous navigation behaviour within the Quanser QLabs simulation environment. The simulation framework enabled safe, repeatable testing of perception and vehicle control algorithms prior to physical deployment.

Physical QCar Platform

Practical implementation and experimental validation using the Quanser QCar platform, supporting real-world evaluation of perception and navigation strategies.

Real-Time Object Identification (Physical Vehicle Testing)

Conducted physical vehicle trials using computer vision techniques for real-time object identification during QCar operation.

Deep Learning-Based Cone Detection & Control Pipeline

Developed a deep learning model using YOLO, trained on a cone dataset for detection tasks. Model inference was executed on a PC, with control commands transmitted via a client–server architecture.
  • Deep learning model: YOLO (cone detection)
  • Training dataset: Cone dataset
  • Runtime inference: PC-based execution
  • Control architecture: Client–server command transmission

Key Contributions

  • Built a simulation-to-real autonomous navigation workflow using QLabs and QCar.
  • Validated perception and control algorithms across simulation and physical platforms.
  • Implemented computer vision techniques for real-time environmental perception.
  • Developed a YOLO-based cone detection model integrated with PC-based inference.
  • Designed a client–server architecture for real-time vehicle command transmission.
